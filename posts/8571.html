如何使用LaTex制作演示文稿

LaTex Beamer PPT
LaTex
发布日期:   2020-10-17
更新日期:   2020-10-18
文章字数:   490

近期需要实验室内部学术交流，一直苦于无法做一个适合PPT。之前听师兄提到过使用LaTex也可以做演示文稿，所以我决定尝试一下，写篇博客做个记录。

beamer是什么

无论是使用PowerPoint还是WPS制作演示文稿，最大的痛点莫过于排版问题，而LaTex把排版的问题交给了底层的编译器，能够让创作者把更多的精力放在内容上面，而不是格式上面。

beamer是LaTex中的一个专门制作演示文稿的文档类。使用beamer能够快速地制作一个简洁、美观的演示文稿，非常适合于学术报告。

我的beamer自定义模板

\documentclass[aspectratio=169]{beamer}


% ----------------宏包-----------------
\usepackage{ctex}
\usepackage{bookmark}
\usepackage{booktabs}
\usepackage{graphicx}
\usepackage{subfigure}
\usepackage{multicol}
\usepackage{arev}
\usepackage{hyperref}


% --------------主题------------------
\usecolortheme{rose}    

% -------------修改主题---------------
\setbeamertemplate{navigation symbols}{}    % 不显示导航栏
\useinnertheme{default}
\setbeamertemplate{itemize items}{$\pencil$}
% \setbeamercovered{transparent=7}   % 半透明效果

\setbeamertemplate{headline}{}  % 显示headline
\setbeamertemplate{footline}[frame number] % footline中只显示页码

\definecolor{mygray}{RGB}{245, 245, 245}  % rgb: 取值范围为[0, 1]; RGB: 取值范围为[0, 255]
\definecolor{mygreen}{RGB}{48, 128, 20}  % rgb: 取值范围为[0, 1]; RGB: 取值范围为[0, 255]
\definecolor{mydefault}{rgb}{0.2, 0.2, 0.7}  % rgb: 取值范围为[0, 1]; RGB: 取值范围为[0, 255]

\setbeamercolor{title}{bg=mygray}


% ------------------------------------
\begin{document}  % \renewcommand\contentsname{目录}
\title{如何入门一场机器学习比赛}
\subtitle{————DIGIX 精英大赛总结}
\author{王钦正（研二上）}
\date{2020/10/16}

% -------------标题页----------------
\begin{frame}[plain]
    \titlepage
\end{frame}

\begin{frame}{目录}
    \tableofcontents[hideallsubsections]
\end{frame}

% -------------------比赛意义----------------------
\section{比赛意义}

%  此处省略一万字
%  。。。。。。。
%  。。。。。。。

% ------------------前期准备-----------------------
\section{前期准备}
\begin{frame}{目录}
    \tableofcontents[currentsection, hideothersubsections]
\end{frame}


%  此处省略一万字
%  。。。。。。。
%  。。。。。。。
\end{document}
